# Vend

Buying / Selling platform made with React and Golang.

## Structure

- Poutine (Frontend)
  - TypeScript
  - React
  - TailwindCSS
- Plantain (API/Backend)
  - Golang
  - Fiber (web server)
  - Prisma
  - PostgreSQL
